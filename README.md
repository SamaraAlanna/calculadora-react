# Calculadora React

Projeto desenvolvido como parte do Bootcamp da [DIO (Digital Innovation One)](https://www.dio.me/), utilizando React com componentes funcionais e hooks customizados.

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Porcentagem e troca de sinal (+/-)
- Encadeamento de operações sem precisar apertar `=`
- Display com expressão anterior e valor atual
- Operador ativo destacado visualmente
- Suporte a teclado físico

## Tecnologias

- React 18
- CSS Modules
- Hook customizado (`useCalculator`)
- Create React App

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/SamaraAlanna/calculadora-react.git

# Entre na pasta
cd calculadora-react

# Instale as dependências
npm install

# Rode o projeto
npm start
```

Acesse [http://localhost:3000](http://localhost:3000) no navegador.
